Q1:What is the web development and what are types of it?

ANS: Web Programming (or web development) is the art of constructing and maintaining web pages for websites by a web developer
(also known as a web engineer) to give the user what he needs from the website
the type of it three type
1 - frontEnd
2 - backEnd
3 - fullStack

Q2:What is difference between web developers and web designers?

ANS: Web developers focus on the technical aspects and functionality of a website, using programming languages,
while web designers concentrate on the visual and user experience, using design tools and principles.
Developers code and make websites work, while designers create the look and feel of websites.

Q3:Is the best for web developer to be full stack development or just one type of development?

ANS: It depends on your career goals and interests. Full-stack development offers versatility,
while specializing in front-end, back-end, mobile, or niche areas can lead to expertise in a specific domain.
